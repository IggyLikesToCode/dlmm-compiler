# Executor Deployment

Added interactive deployment to the TypeScript executor so it can actually deploy strategies on-chain instead of just previewing them.

## What Changed

- executor.ts now loads wallet from .env file (PRIVATE_KEY)
- Asks for confirmation before deploying each strategy
- Retries failed transactions once before stopping
- CLI accepts pool address and token amounts as arguments

## New Commands

Preview (no wallet needed):
```
npx ts-node src/sdk/executor.ts strategy_plan.json --pool <address> --preview
```

Deploy (needs PRIVATE_KEY in .env):
```
npx ts-node src/sdk/executor.ts strategy_plan.json --pool <address> --amount-x 1000000000 --amount-y 1000000
```

## Files Changed

- src/sdk/executor.ts - added wallet loading, prompts, retry logic, CLI args
- src/sdk/client.ts - createPosition now returns position key
- .env.example - template for private key

## Dependencies Added

- dotenv
- bs58
